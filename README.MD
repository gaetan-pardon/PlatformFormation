
# Platform de formation



## instalation 


 

Aller dans backend 

 

Pour utiliser le docker vous devez fournir les varables de enviroment normalment présent dans le .2.env (Le ficher .2.env.dummy vous est fourni comme model) 

 

 

Si vous n’utilisez pas le conteneur   

Lancer l’installation des dependance : 

' Python –m pip install –r requirement.txt  

 

Puis vous pouvez  aux choix 

 

Fournir à l’éxecution les varables d’ environement normalement présent dans le .2.env (le ficher .2.env.dummy vous est fourni comme model) 

 ou  

Fournir dans le répertoire un fichier .env contenant ces derniers (le ficher .env.dummy vous est fourni comme model)
